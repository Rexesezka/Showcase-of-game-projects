# Showcase of Game Projects - Backend

Django проект для загрузки и запуска Unity игр в браузере.

## Установка

1. Перейдите в папку преокта:
```bash
cd backend
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Выполните миграции базы данных:
```bash
python manage.py migrate
```

4. Запустите сервер разработки:
```bash
python manage.py runserver
```

5. Откройте браузер и перейдите на `http://127.0.0.1:8000/`

## Использование

1. На главной странице вы можете загрузить ZIP архив с Unity WebGL билдом игры
2. После загрузки игра появится в списке доступных игр
3. Нажмите "Играть" чтобы запустить игру в браузере

## Формат архива

Архив должен содержать Unity WebGL билд игры. Обычно это включает:
- `index.html` - главный HTML файл
- Папка `Build/` с JS и WASM файлами
- Папка `TemplateData/` с ресурсами

Если `index.html` находится в подпапке, система автоматически переместит файлы на нужный уровень.

## Структура проекта

```
backend/
├── game_showcase/      # Основной проект Django
│   ├── settings.py     # Настройки проекта
│   ├── urls.py         # Главные URL маршруты
│   └── wsgi.py         # WSGI конфигурация
├── games/              # Приложение для управления играми
│   ├── models.py       # Модели данных
│   ├── views.py        # Представления (views)
│   ├── urls.py         # URL маршруты приложения
│   └── templates/      # HTML шаблоны
├── media/              # Загруженные файлы (создается автоматически)
│   └── games/          # Загруженные игры
└── manage.py           # Django управляющий скрипт
```
